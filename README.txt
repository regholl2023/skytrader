SkyTrader

SkyTrader is a simple trading bot that implements a trend-following strategy using technical indicators such as Moving Averages, MACD, RSI, and Bollinger Bands. This bot is designed to execute buy and sell orders for a given stock symbol based on the signals generated by the selected indicators.
Features

    Utilizes historical market data to generate trading signals.
    Implements Moving Averages Convergence Divergence (MACD) for trend confirmation.
    Utilizes Relative Strength Index (RSI) for identifying overbought and oversold conditions.
    Implements Bollinger Bands for assessing volatility and identifying potential entry and exit points.
    Optimizes strategy parameters to maximize returns using parameter optimization techniques.

Prerequisites

    Python 3.x installed on your system.
    Alpaca API credentials (API key and secret key) to access market data and execute trades.
    Required Python packages: pandas, alpaca_trade_api, numpy, talib, datetime.

How to Run

    Clone the repository to your local machine.
    Install the required Python packages using pip install -r requirements.txt.
    Set up your Alpaca API credentials by creating a config.ini file in the project directory and adding your API key, secret key, and base URL.

    ini

    [alpaca]
    api_key = YOUR_API_KEY
    secret_key = YOUR_SECRET_KEY
    base_url = https://paper-api.alpaca.markets  # Use the paper trading base URL for testing

    Adjust the stock symbol, start date, end date, and timeframe as needed in the skytrader.py script.
    Run the skytrader.py script.
    Monitor the output for transaction logs and total return.
